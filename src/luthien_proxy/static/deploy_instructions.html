$ Connect Claude Code

Point Claude Code at this proxy. One environment variable, then you're ready.

1

Set the proxy URL

Run this in your terminal before launching Claude Code:

bash 
export ANTHROPIC_BASE_URL={{BASE_URL}}

Your existing Anthropic credentials (API key or OAuth) pass through automatically.

Note: Do not append /v1 to the URL — the Anthropic SDK adds it automatically.
2

Launch Claude Code

Start Claude Code as you normally would:

bash 
claude

All requests will now route through the proxy.

3

Verify the connection (optional)

Confirm the proxy is reachable:

bash 
curl {{BASE_URL}}/health

You should see {"status": "ok"}. Once Claude Code connects, requests will appear in the Activity Monitor.

Claude Code can't connect

Check that ANTHROPIC_BASE_URL does not include /v1 — the SDK appends it.

Verify the proxy is running with curl {{BASE_URL}}/health.
