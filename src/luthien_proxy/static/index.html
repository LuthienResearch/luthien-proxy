Monitoring UIs

GET /activity/monitor UI
Real-time activity monitor — view all gateway requests, policy events, and responses
GET /policy-config UI
Activate, configure and test policies
GET /credentials UI
Manage authentication modes and cached credentials
GET /diffs UI
Policy diff viewer — compare original vs modified requests/responses
GET /history UI
Conversation history — browse past sessions with full message transcripts

API Endpoints

All API endpoints require admin authentication (ADMIN_API_KEY)

GET /api/debug/calls API
List recent calls with event counts (supports ?limit=N)
GET /api/debug/calls/{call_id}/diff API
JSON diff showing what the policy changed
GET /api/activity/stream SSE
Server-Sent Events stream of all gateway activity
