Luthien Control

AI safety and control for production agentic deployments. See what your AI agents are doing. Enforce the rules you care about.

Why Luthien

Drop-in Proxy

Works with Claude Code, Codex, and any OpenAI/Anthropic-compatible tool. Point your agent at Luthien and you're done.

Real-Time Visibility

Watch every request and response as it happens. See exactly what your AI agent is doing, thinking, and outputting.

Policy Enforcement

Define rules. Luthien enforces them. Block dangerous commands, filter sensitive data, or transform responses on the fly.

Hot-Swap Policies

Change policies without restarting anything. Click to activate a new policy and it takes effect immediately.

Get Started

git clone https://github.com/LuthienResearch/luthien-proxy cd luthien-proxy cp .env.example .env # Add your API keys to .env ./scripts/quick_start.sh # Launch Claude Code through the proxy ./scripts/launch_claude_code.sh

Write Your Own Policy

from luthien_proxy.policies.simple_judge_policy import SimpleJudgePolicy class MyPolicy(SimpleJudgePolicy): RULES = [ "Never allow 'rm -rf' commands", "Block requests to delete production data", ] # That's it. Luthien handles the rest.
